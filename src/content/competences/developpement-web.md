## Définition

Le développement web couvre l'ensemble du cycle de création d'une application accessible via un navigateur : de la conception de l'interface utilisateur (front-end) à la logique métier et à la persistance des données (back-end). En 2025, la frontière entre les deux s'estompe avec les frameworks isomorphes (Next.js, Astro, Nuxt) qui unifient rendu serveur et interactivité client.

En entreprise, le développeur web est un acteur central de la transformation numérique. Chez des acteurs comme 3F, de nombreux processus métier (gestion locative, demandes de réparations, communication avec les locataires) reposent sur des applications web sur mesure.

## Éléments de preuve

### Développement et maintenance chez 3F

Dans mon alternance chez 3F, je développe et maintiens des applications internes en PHP/JavaScript avec des bases de données Oracle. J'ai notamment refactorisé un module de gestion des demandes locataires vieillissant : réduction de la dette technique, mise en place d'une couche de service, et ajout d'une API REST pour les appels asynchrones.

**Résultat** : Le temps de chargement du module a été réduit de 40 %. Les nouveaux développeurs intègrent le code deux fois plus vite grâce à la documentation interne que j'ai produite.

### Let'Style — Application web full-stack

Let'Style est une plateforme web développée de zéro avec gestion des utilisateurs, boutique intégrée et personnalisation vestimentaire. J'ai fait les choix architecturaux (MVC, API REST, authentification JWT) et implémenté les fonctionnalités de bout en bout.

**Résultat** : Application fonctionnelle avec 100 % des fonctionnalités spécifiées dans le cahier des charges, déployée sur un hébergeur cloud.

## Autocritique

C'est ma compétence principale (9/10). Je maîtrise plusieurs langages et frameworks, et je suis capable de travailler aussi bien sur le front que le back. Ma limite actuelle est la maîtrise des architectures distribuées à très grande échelle (microservices, orchestration Kubernetes) — un domaine que j'expérimente mais sur lequel mon expérience pratique reste à consolider.

Cette compétence est transférable à tous les contextes numériques, mais exige une veille technologique continue : un stack pertinent en 2023 peut devenir obsolète en 2025.

Conseil : choisir un framework et l'explorer en profondeur avant d'en changer — la maîtrise d'un outil vaut plus que la connaissance superficielle de dix.

## Évolution

Je continue de progresser sur les frameworks modernes (Astro, SvelteKit) et sur TypeScript pour renforcer la robustesse de mon code. Mon prochain objectif est d'obtenir une certification cloud (Azure AZ-900 ou AWS Cloud Practitioner) pour compléter mon profil full-stack avec des compétences d'hébergement et de déploiement.
